Cognitive Rationale — 이 시각화의 설계 근거

Pattern

PAT-001 (side_by_side) — 본질이 "2→1 통합"이므로 좌우 비교가 가장 효과적. Before에 2개 스택(높이 큼), After에 1개 스택(높이 작음) → 시각적 축소가 통합 효과를 즉시 전달.

Applied Rules

  • RULE-L01 — Before/After 앵커 정렬: 상품 필터 스택이 양쪽에 동일 위치 ← PRC-001
  • RULE-C01 — 색상 언어: 기존(파랑), 삭제(회색+취소선), 신규(초록), 병합(주황) ← PRC-003
  • RULE-C09 — Slate 다크 테마 ← AES-001
  • RULE-T01 — 스택 노트 1문장 요약 ← IP-001
  • RULE-CMP03 — concern 행(Before 리뷰 스택 dashed border)으로 삭제 대상 사전 예고 ← ATT-003

Self-Evaluation

  • change_visibility: 5/5 — 스택 높이 비대칭 + 색상 코딩 + dashed border + Insight 배너

  • text_economy: 5/5 — 스택당 1문장 노트, 파일명은 코드 그대로

Evidence Trail
