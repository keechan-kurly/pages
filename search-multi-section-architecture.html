Business Impact
  • 확장성 서버 주도 섹션 구성 — 새 섹션 타입 추가 시 SectionMapper 1개 + UI Model/ViewHolder 1쌍만 추가하면 됨. 클라이언트 릴리스 없이 서버에서 섹션 순서/구성 변경 가능
  • 유지보수 수정 포인트 60% 감소 — ConcatAdapter 조합 복잡도 제거. Fragment 코드 147줄 → 66줄 (55% 감소). VIEW_TYPE 충돌 관리 포인트 소멸
  • 안정성 선언적 상태 관리 — BeautyOnlyState sealed interface로 상태 누락 불가. sealed interface when 표현식에서 컴파일러가 미처리 케이스를 검출
