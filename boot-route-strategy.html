Pattern
PAT-024 — 데이터 흐름도 (Data Flow Diagram)
Boot API 응답에서 라우팅 결정까지의 데이터 변환 파이프라인과, BootRouteResolver.resolve()의 결정 트리를 시각화. 리뷰어가 “이 플래그 조합에서 어떤 라우트가 나오는가?”를 코드 읽기 전에 파악할 수 있도록 설계.
