패턴
PAT-024 — 데이터 흐름도 (Data Flow Diagram)
3개 PR이 CI 파이프라인의 서로 다른 단계에 위치하므로, 데이터(PR 이벤트)가 각 단계를 거치는 흐름으로 표현. EVAL-0006 편향 수정 후 첫 적용 — Step 1에서 "데이터 흐름"으로 판별, PAT-009/026 대신 PAT-024 선택. 수직 타임라인 + 카드 내부 수평 플로우의 이중 구조로 전체 진행과 단계별 상세를 동시에 전달.
적용 규칙
  • RULE-L01 수직 타임라인이 4단계의 앵커 역할 — 시선이 위→아래로 자연스럽게 흐름 ← WM-001
  • RULE-C01 PR별 고유 색상 — 초록(#7235) / 파랑(#7271) / 핑크(#7326) — 시각적 귀속 즉각 인지 ← PRC-003
  • RULE-C09 Slate 다크 테마 기본 적용 ← PRC-004, AES-003
  • RULE-T01 각 카드에 1문장 pipe-desc로 핵심 요약 ← IP-001
  • RULE-CMP03 Hero 영역에 3개 PR 배지로 사전 안내 (무엇을 볼 것인지 예고) ← ATT-002
  • EVAL-0006 PAT-024 첫 사용 — 3단계 패턴 선택 편향 수정 적용 ← DM-001
