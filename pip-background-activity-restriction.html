PiP 모드 Activity 시작 타이밍 변경

Samsung One UI 보안 패치로 백그라운드 Activity 시작이 제한되어, 라이브커머스 PiP 모드에서 상품 상세 화면 전환이 차단되는 이슈를 해결한다.

왜 변경했는가?
Samsung One UI 보안 패치가 백그라운드 Activity 시작 제한 정책을 강화했다. 기존 방식은 PiP 전환 완료(= 백그라운드 상태) 후 startActivity를 호출하므로, 보안 정책에 의해 상세 화면 전환이 차단된다. 포그라운드 상태에서 먼저 Activity를 시작하면 정책에 걸리지 않는다.
